# **4ï¸âƒ£ `clientController.js` (Handles API Requests)**
### **Purpose:** This file handles API requests and sends responses to frontend.

```js
import * as clientService from "../services/clientServices.js";

export const getClients = async (req, res) => {
    try {
        const clients = await clientService.getClients();
        res.status(200).json(clients);
    } catch (err) {
        console.error('Error fetching clients:', err);
        res.status(500).json({ message: "Internal Server Error" });
    }
};
```

### **Functionality:**
âœ… Calls `getClients()` from `clientServices.js`
âœ… If successful, **sends client data as JSON** (`res.json(clients)`)
âœ… If there's an error, **returns a 500 error message**
